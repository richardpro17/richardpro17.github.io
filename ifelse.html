CONDICIONALES ESTRUCTURAS IF/ELSE

Tendras toda la documentacion mejor seleccionada para ti de la estructura condicional if else.


El primer paso para entender las estructuras IF/ELSE es saber como funcionan, las declaraciones condicionales en programacion se usan para tomar decisiones basadas en las condiciones, estas se ejecutan secuencialmente cuando no hay ningun condicion alrededor.


La estructura if-else es un tipo de estructura booleana, donde se completa una accion especifica si se completa la condicion en verdadera


CONCEPTOS DE ESTRUCTURAS CONDICIONALES:


IF: Es para especificar un bloque de codigo de que pasara si la condiciones es verdadera.


ELSE: para especificar un bloque de codigo a ejecutar si la condicion no se cumple y es falsa


ELSE IF: para especificar una nueva condicion para probar, si la primera condicion es falsa pasara al "Else"


SWITCH: para especificar muchos bloques de alternativos de codigo para ejecutar


FORMA DE DECLARACION:
if (Condicion){
//Bloque de ejecucion si la condicion es verdadera


if (10>5){
   printf("10 es mayor a 5");
}
EJEMPLO ELSE IF:
numero=30
if (numero>50){
   printf(numero"Es mayor que 50");
} else{
   printf(numero" Es menor que 50")
}
EJEMPLO SWITCH:
switch( c ){
  case 'A':
      printf("Es una letra A");
  case 'a':
      printf("Es una letra A");
  default :
   total++;
  }
