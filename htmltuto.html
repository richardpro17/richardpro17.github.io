¿QUE ES HTML?

HTML no es un lenguaje de programacion, suena raro verdad, pues el ("Hypertext Markup Language") es un lenguaje de marcado, en resumen es un lenguaje que le dice a los navegadores web como estructuras la pagina que nuestros ojos ven.

HTML consiste en crear una serie de elementos que ayudan a las diferentes partes del contenido para que tu decidas de que forma aparecer y actuar.

PARTES BASICAS DE UN DOCUMENTO HTML


La mayoria de paginas web siempre mantienen una formalidad en sus estructura para que sea facil, accesible para los desarrolladores y la experiencia de usuario.


Encabezado: La parte de arriba que por lo general contiene el titulo de la pagina de forma impresionante, logo de tu marca etc, inclusive un lema, no suele alterarse dentro de la misma pagina web.


Menu de navegacion: La parte donde se acomoda desde un buscador, un menu interactivo para que los usuarios naveguen de forma sencilla y rapida dentro del sitio web.


Contenido principal: Es la gran parte de informacion que abarca la mayoria de la pagima, esta parte siempre esta en constante cambio dependiendo los subtemas o temas del sitio.


Barra Lateral: La barra lateral se usa como menu de navegacion o tambien algun tipo de informacion adicional, publicida etc, la mayoria de veces el uso que se la esta relacionado con el contenido principal mostrado en ese momento.


Pie de pagina: el pie de pagina la mayoria de veces esta hasta abajo, contiene letras pequeñas e informacion relevante de contacto del autor o marca, no suele ser tan importante pero forma parte del contenido de una pagina web.

Esquematizacion basica de un documento HTML:


Un archivo html siempre empieza con la etiqueta HTML.
Por otra parte, todo el documento HTML consta de dos partes principales, la cabecera HEAD(head) y el cuerpo del documento (body).

1.-La cabecera contiene informacion destinada para el navegador que queda oculta al usuario. Su etiqueta es "HEAD" encerrado en <"Head>".


2.-El cuerpo del documento Body es el que el usuario puede ver si etiqueta es <"Body>".
