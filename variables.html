Variables

Aprenderas lo basico como el concepto de variables y su uso, tipos de estas para usarla en los lenguajes de programacion de forma global sin tener que volver a repasarlas.


Para empezar una variable se define como una caracteristica o condicion que esta sujeta a siempre tener cambios, la variable se usa para asumir el valor especifico de uno o varios individuos.
Estas variables esta formada por un espacion en el sistema de almacenaje de un ordenador y un identificador para nombrar esta variable.


TIPOS DE VARIABLES Y COMO DECLARARLAS:


  • 1.-Numerica: en estas variables como su nombre indica, podemos almacenar numeros por ejemplo:
    int numero = 10
  • 2.-Cadena de texto: en el caso de estas variables se almacena texto al estar entre comillas. Hay de longitud fija y de longitud variable por ejemplo:
    char str[] = "Pro"; palabra = "Hola";
  • 3.-Booleanas: tienen un valor logico el cual es "true" o "false", otra forma de representacion de estos valores es por medio del 0 y el -1.
  • 4.-HTML: estas variables se identifican por medio de su (id), estas requieren un selector para asignar los elementos a la variable.
  • 5.-Arreglos: Estas variables sirven para almacenar mas de un elemento junto con otras varialbes, solo requieres redactarlo con corchetes y separarlos con una coma como en este ejemplo:
    Numeros[]= (2, 5, 10, 32, 100)
  • 6.-Objetos: Es para agregar objetos y asignarles sus propiedades , la diferencia de esta es la forma en la que se declara por ejemplo
    {color: verde}
